## Productivity Measurement Context
- The RHOAIENG Jira project tracks engineering work (stories, bugs, tasks, sub-tasks)
- Each associate's work should be queried via their Jira display name as assignee
- Key metrics: issues resolved over time, story points completed, average cycle time
- Issues have statuses: New, Backlog, In Progress, Review, Testing, Resolved, Closed
- "Done" = Resolved or Closed
- Time periods to track: weekly, monthly, quarterly rolling windows
